4Orbs

What is 4Orbs?

4Orbs is a proof of concept of a new way to display the scores from 8values-style quizzes.

This version features questions from both 8values and NewValues.

Feel free to clone the source code on github and create your own version for different quizzes that use the same format.

What if I don't like my results?

All the questions in this test were made by the authors of 8values and NewValues, I suggest you forward any complaints to them.
